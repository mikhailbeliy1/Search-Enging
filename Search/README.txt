This project is a small search engine and stores data of documents in files
To update the data run these files in order:
    1. Crawler/python files/crawler.py (To change the number of pages crawled refer to Crawler/python files/constant.py)
    2. Crawler/python files/n_grams.py
    3. Clusters/python files/doc_vector_maker.py
    4. Clusters/python files/cluster_maker.py
    5. PageRank/python files/page_rank.py
    6. HITS analysis/python files/hits.py
